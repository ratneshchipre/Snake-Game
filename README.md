# Snake Game

A classic snake game implementation with smooth controls, score tracking, and mobile-friendly touch controls.

## Features
- Classic snake gameplay mechanics
- Keyboard controls (Arrow keys)
- Touch-friendly mobile controls
- Real-time score tracking
- High score persistence (Local Storage)
- Automatic game speed (125ms interval)
- Game over detection (wall collision & self-collision)
- Visual snake and food indicators

## Technologies Used
- HTML5
- CSS3 (Grid Layout)
- Vanilla JavaScript
- Font Awesome Icons

## Installation
1. Clone the repository
```bash
   git clone https://github.com/ratneshchipre/Snake-Game.git
```
2. Navigate to the project directory
```bash
   cd Snake-Game
```
3. Open `index.html` in your browser
